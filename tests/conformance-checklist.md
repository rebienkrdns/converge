# Conformance Checklist

- The repository defines the permanent council.
- The repository defines the adaptive council.
- Each engine has inputs, outputs, responsibilities, and interaction notes.
- Each gate defines purpose, approval conditions, blocking conditions, and evidence requirements.
- The decision cycle includes all required phases.
- Self critique covers assumptions, blind spots, missing evidence, counterarguments, confidence, and future risks.
- The spec explains the canonical decision model.
