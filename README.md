# Converge

> **No escribimos código primero. Tomamos decisiones primero.**

Converge es un Engineering Intelligence Framework para estructurar la deliberación técnica antes de implementar cambios.

## Estructura
- `docs/`: filosofía y principios
- `spec/`: especificación abierta
- `framework/`: componentes del framework
- `packs/`: extensiones por tecnología
- `examples/`: ejemplos de uso
